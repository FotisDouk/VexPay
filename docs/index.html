VexPay

Self-hosted, non-custodial crypto payments — fast, secure, and easy.

Early development

VexPay is a payment gateway you run on your own server. Point it at your own wallet, get an API key, and start accepting cryptocurrency. Funds go directly to your wallet — VexPay never holds your private keys or your money.

Quick start

git clone https://github.com/vexarnetwork/vexpay
cd vexpay
go run ./cmd/vexpay

curl http://localhost:8080/healthz
